## 二、图片标签

img标签：

必须要属性：src，图片的路径

​	绝对路径：从根盘符开始，网络地址

​	相对路径：参照物，相当对当前的html文件

非必须要属性：

​	width：宽度

​	height：高度

​	border：边框线

​	alt：图片不显示时，的文字

​	title：鼠标在图片上，的文字

​	align：位置

​		left，right，没有center

​		top，middle，bottom



## 三、列表

块级元素：另起一行

### 3.1 有序列表

ol：order list

li：list

```html
<ol>
  <li></li>
  <li></li>
  ...
</ol>
```

属性：type：1，a，A，i，I，start



### 3.2 无序列表

ul：un order list

li：list

```html
<ul>
  <li></li>
  ...
</ul>
```

属性：type：disc(实心圈)，circle(空心)，square(正方形)



### 3.3 自定义列表

dl：

dt：

dd：

```html
<dl>
  <dt></dt>
  <dd></dd>
</dl>
```

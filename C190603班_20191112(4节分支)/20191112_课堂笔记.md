## 三、总结

选择结构：

if语句：

A：简单的if语句：条件满足就执行语句1；

```
if（条件）{
  语句1；
}
```

B：if。。else语句：条件满足执行语句1，否则执行语句2：二者必须其一

```
if(条件){
  语句1；
}else{
  语句2；
}
```

C：if的嵌套：多选一

```
if（条件1）{
  语句1；
}else {
  if（条件2）{
    语句2；
  }else{
    。。。
  }
}
简写：
if（条件1）{
  语句1；
}else if（条件2）{
  语句2；
}else if（条件3）{
  语句3；
}。。。
else{
  
}
```



switch语句：

```
switch（表达式）{
  case 数值1：分支1；break；
  case 数值2：分支2；break；
  。。。
  default：
}
```

switch后的表达式，和case后的数值进行匹配，全等的匹配：数值和类型，匹配上哪个case就执行哪个分支语句，直到break位置。如果所有的case都没有匹配上，可以执行default语句。可选的。
